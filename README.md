# Requirements

* Node/npm >= 8 
* React Native


# Development

Install Dependencies 

```sh
npm install
```


Link native module (randombytes in our case)
```sh
react-native link
```

RN-nodefy hack

```sh
npm run postinstall
```

## Run local app

```sh
react-native run-android 
```

```sh
react-native run-ios
```
